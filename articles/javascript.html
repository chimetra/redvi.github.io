09 Октябрь 2012

Что представляет из себя сам JavaScript?

По сути своей это скриптовый язык программирования. В подавляющем большинстве случаев используется при веб-разработке для придания интерактивности веб-страницам.

JavaScript работает на стороне клиента — браузера — а не на сервере. Может способствовать как созданию более привлекательных или же функциональных веб-страниц, так и появлению уязвимостей. А посему будьте аккуратны при написании JS-скриптов или при возможности откажитесь от них вовсе.

Циклы while и for

WHILE — пока.

Пока переменная a меньше 20, увеличивать её на два и выводить её значение на экран

var a = 2;
while ( a < 20)
    {
        a++;
        document.write(a);
    }

do while

var a = 2;
do
     {
         a++;
         document.write(a);

     }
while (a < 20);

Отличие do while от while: do while выполнит цикл хотя бы один раз

А вот пример скрипта, который часто используется на «вредоносных» страницах, результат его выполнения — пользователь не может просто так закрыть всплывающее окно. Назовём этот скрипт «Достань пользователя» :)

do {

var s = prompt("Введите своё имя", "");
}
while (!s)

document.write("Привет, " + s + " !");

Итак, выполняется цикл с предусловием — do while — у пользователя запрашивается некая информация, условие гласит: выполнять цикл до тех пор, пока значение не равно s — not s, т.е. — (!s) , то есть бесконечно.

Объекты в Java Script

Объект содержит в себе данные и функции, обрабатывающие эти данные. Данные называются свойствами объекта, а функции — методами.

var myBook = new myBook(); // создание объекта
myBook.title = "название"; // операция разименования
myBook.price = 200;
myBook.show ();

myBook = null; // уничтожение объекта
